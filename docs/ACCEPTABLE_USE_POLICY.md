# Averion Acceptable Use Policy
Last updated: June 2026

---

## 1. Purpose

This policy defines acceptable use of the Averion
platform to protect all users and the platform.

---

## 2. Prohibited Activities

You must NOT:

Technical abuse:
→ Attempt to reverse engineer any part of the platform
→ Attempt unauthorized access to other accounts
→ Use automated scripts to abuse the API
→ Perform denial of service attacks
→ Attempt to bypass security measures
→ Exploit software bugs for personal gain
  (report bugs to support@averionbot.com instead)

Account abuse:
→ Share your account with other users
→ Create multiple accounts to avoid fees or limits
→ Create fake referral accounts
→ Impersonate other users or Averion staff

Financial abuse:
→ Attempt to reverse legitimate transactions
→ Dispute valid performance fees
→ Use stolen exchange API keys

Illegal activities:
→ Use the platform for money laundering
→ Use the platform for any illegal purpose
→ Trade on behalf of sanctioned persons or entities

---

## 3. Consequences

Violations may result in:
→ Immediate account suspension
→ Suspension of withdrawals from reserve wallet pending investigation
→ Reporting to relevant authorities
→ Legal action for damages

---

## 4. Reporting

To report abuse: support@averionbot.com
To report security vulnerabilities: same email
We investigate all reports seriously.

---

## 5. Changes

We may update this policy at any time.
Changes effective immediately upon posting.
