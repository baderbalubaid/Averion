# Averion Privacy Policy
Last updated: June 2026
GDPR compliant · EU server

---

## 1. Who We Are

Averion ("we", "us", "our") operates averionbot.com.
Server location: Hetzner · EU data center.
Contact: support@averionbot.com

---

## 2. What We Collect

Account information:
→ Username
→ Email address
→ Phone number (only if verified by admin request)
→ Password (bcrypt hashed · never plain text)

Trading information:
→ Exchange API keys (Fernet encrypted · never plain text)
→ Trade history and positions
→ Bot configurations and parameters
→ Reserve wallet transactions

Technical information:
→ IP address
→ Device fingerprint (browser/app info)
→ Login timestamps
→ Session tokens (hashed)

## 5. API Key Security

Exchange API keys are:
→ Encrypted using Fernet symmetric encryption
→ Split key stored in two separate secure locations
→ Never stored or transmitted in plain text
→ Not visible to Averion staff in plain text
→ Rotated monthly (encryption key rotation)

Trade-only permissions required.
Withdrawal permissions must be disabled.

---

## 6. Data Storage & Backups

Primary storage: Hetzner EU server (PostgreSQL)
Backups: daily encrypted backup · retained 7 days
Backup location: same Hetzner data center
Backup encryption: same Fernet encryption

## 9. Your Rights (GDPR)

You have the right to:
→ Access: request a copy of your data
→ Rectification: correct inaccurate data
→ Erasure: delete your account and data
→ Portability: export your trade history (CSV)
→ Restriction: limit how we process your data
→ Object: opt out of non-essential processing

To exercise rights: support@averionbot.com
Response within 30 days.
